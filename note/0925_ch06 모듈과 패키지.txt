ch 06 모듈과 패키지

1.  파이썬 모듈 사용
	(1) import A		-A 모듈 가져옴

	(2) from A import B 	-A 패키지 안의 B 모듈 가져옴
	    from A.B import C	-A안의 B모듈 안의 C함수 가져옴

	(3) import A as B		-A모듈을 B이름으로 가져옴

	(4) from A import B as C	- A 패키지 안의 B 모듈을 C이름으로 가져옴
	    from A.B import C as D	- A안의 B 모듈 안의 C 함수를 D이름으로 가져옴
