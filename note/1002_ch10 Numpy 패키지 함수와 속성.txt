Numpy 패키지의 함수와 속성

	- 배열속성
		dtype: 요소들의 type
		itemsize : 요소의byte사이즈
		ndim: 차원 수(축 수)
		size: 요소 갯수
		shape: 배열의 구조(튜플)

	-배열 생성 함수
		np.array(obj, dtype= np.int64 OR 'int64', copy=T/F)
		np.zeros(tuple,dtype)
		np.ones(tuple, dtype)
		np.full(tuple, 기본값, dtype)
		np.empty(tuple, dtype)
		np.random.random(tuple) : 0-1미만의 균등분포 랜덤값 
		np.random.uniform(low=0,high=1.0,size=tuple) low~high 미만 균등분포 랜덤값 (float, int 둘 다 low, high 값으로 가능)
		np.random.normal(loc=0, scale=1 size=tuple) 평균 loc, 표준편차 scale 정규분포 랜덤값
		np.arange(from, to, by, dtype): from 부터 to 앞까지 by씩 건너 뛴 값으로 1차원 배열 

	- 배열차원변경
		reshape() : -1 사용하면 자동계산
		resize() : -1 사용 불가
		T OR transpose() : 전치행렬
		ravel(), flatten(): N차원을 1차원으로 변경
